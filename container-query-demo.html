5. 配置要点

Tailwind 配置

// tailwind.config.js
module.exports = {
  plugins: [
    require('@tailwindcss/container-queries'),
  ],
}

CSS 容器类型

/* 宽度查询 */
container-type: inline-size;

/* 高度查询 */  
container-type: size;

/* 命名容器 */
container-name: my-container;
