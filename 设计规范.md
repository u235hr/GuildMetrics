# 项目技术与设计规范

## 1. 核心设计原则

本规范旨在确保项目在视觉与技术实现上的一致性、可维护性和高质量。

- **容器自适应**: 所有组件和元素都应基于其父容器的大小进行等比缩放，而非整个视口。
    - **实现**: 在组件根元素上使用 `container-type: size` 开启容器查询上下文。
- **相对单位**:
    - **字体**: 使用容器查询单位 `cqw` / `cqh` 作为根 `font-size` 基准，例如 `text-[min(1.8cqw,1.8cqh)]`。
    - **尺寸与间距**: 组件内部所有尺寸（`width`, `height`）、间距（`gap`, `padding`）、装饰（`border-radius`）等均使用 `em` 单位，以继承基准字体大小，实现等比缩放。
- **严格约束**:
    - **禁止固定像素**: 严禁在 CSS 中使用 `px` 单位，以防破坏自适应布局。
    - **禁止内联样式**: 样式应通过 Tailwind CSS 功能类或外部 CSS 文件定义，保持代码整洁。
    - **禁止溢出**: 任何元素都不能超出其父容器的边界。

---

## 2. 技术栈 (Technology Stack)

- **框架 (Framework)**: Next.js 14+
- **样式 (Styling)**: Tailwind CSS (with Container Queries plugin)
- **状态管理 (State Management)**: Zustand
- **动画 (Animation)**: React CountUp

---

## 3. 整体视觉风格

- **主题**: 结合现代科技感与数据权威性，营造专业、清晰、富有荣誉感的视觉体验。
- **配色**:
    - **主背景**: 深色渐变 (`#1A1A2E` -> `#0F0F1B`)。
    - **容器背景**: 半透明深黑色 `rgba(0, 0, 0, 0.2)`，结合 `backdrop-filter: blur(4px)` 实现毛玻璃效果。
    - **核心元素**: 金 (`#FFD700`)、银 (`#C0C0C0`)、铜 (`#CD7F32`) 金属质感渐变。
    - **交互色**: 激活状态为半透明蓝色 `rgba(59, 130, 246, 0.5)`。
    - **文字色**: 主文字为白色 (`#FFFFFF`)，次要文字为 `rgba(255, 255, 255, 0.7)`。
    - **指示色**: 增长为红色 (`#EF4444`)，减少为绿色 (`#22C55E`)。
- **字体**: `Inter` (Next.js 默认)。

## 4. 页面与组件布局规范

### 4.1. 总体布局

- **全屏布局**: 采用 `flex flex-col` 布局，占满整个屏幕 (`h-screen`)，组件间有 `1rem` (`gap-4`) 的间距。

### 4.2. Header 导航

- **尺寸**: 占父容器高度 `10%` (`h-[10%]`)。
- **字体基准**: `text-[min(1.8cqw,1.8cqh)]`。

### 4.3. Ranking (排行榜) 页面

#### 4.3.1. Top 3 排名卡片

- **尺寸**: 占父容器高度 `30%` (`h-[30%]`)。
- **字体基准**: `text-[min(2cqw,2cqh)]`。

#### 4.3.2. Rest-Ranking (其余排名) 表格

- **尺寸**: 占父容器高度 `55%` (`h-[55%]`)。
- **字体基准**: `text-[min(1.6cqw,1.6cqh)]`。
- **功能**: 表头固定，内容区域可垂直滚动。
- **列定义与宽度**:
    - `排名` (10%)
    - `主播名` (30%)
    - `礼物价值` (25%)
    - `月度环比` (17.5%)
    - `同比增长` (17.5%)

## 6. 数据处理

**注意：当前实现使用 `src/data/testData.ts` 中的模拟数据。以下基于 Excel 的数据处理流程是未来的设计目标，将在数据维护页面开发时实现。**

- **数据源**: 按月命名的 Excel 表（如 `data-2025-09.xlsx`）。
- **头像**: 统一使用占位符 `/public/globe.svg`。
- **增长计算**: 为计算“环比”和“同比增长”，API 路由在获取当月（`M`）数据时，应**同时尝试**读取上一个月（`M-1`）和去年同月（`M-12`）的 Excel 文件。
