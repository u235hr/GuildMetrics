# 月度礼物值排名数据标准格式

## 格式说明
为了确保数据能够正确解析，请按照以下格式整理月度数据：

### 方式一：完整格式（推荐）
```markdown
# 7月币量排名

| 排名 | 主播 | 7月币量 |
|------|------|--------|
| 1 | 主播A | 1,234,567 |
| 2 | 主播B | 987,654 |
| 3 | 主播C | 765,432 |
| **合计** | | **3,987,653** |
```

### 方式二：简化格式
```markdown
| 排名 | 主播 | 7月币量 |
|------|------|--------|
| 1 | 主播A | 1234567 |
| 2 | 主播B | 987654 |
| 3 | 主播C | 765432 |
```

## 格式要求

1. **文件命名**：建议使用 `X月币量排名.md` 格式
2. **表格结构**：必须包含「排名」、「主播」、「X月币量」三列
3. **数据格式**：
   - 排名：纯数字
   - 主播：主播名称
   - 币量：数字，可以包含逗号分隔符
4. **月份识别**：从文件名或标题中自动识别月份

## 数据处理说明

- ✅ 系统会自动校正排名顺序（以礼物值为准）
- ✅ 自动计算总礼物值、平均值、中位数等统计数据
- ✅ 支持环比分析和前三名激励展示
- ✅ 数据验证和错误提示
